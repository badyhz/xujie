1) 项目定位与电梯稿

定位升级：从“人格测评工具” → “人-队-项三阶拓扑共振平台”。用 R 指数统一个人、团队、项目的节奏与可行性，红/黄/绿一眼评估“能不能跑、何时放量”。

电梯稿（15s）：我们把个人、团队、项目放到一张图上，用 R 指数做节奏灯：红线不过不扩张、尖刀不到不冲锋。两周能把乱局拉稳并上线一个闭环，六周推动续费和回款，90 天可规模复制。

2) 核心方法论与 R 指数

TRT｜三阶拓扑共振

将 个人（P）、团队（T）、项目（J）三层量化，同域对齐。

EPM｜等势面匹配

用“项目需求等势面”与“团队供给等势面”求最小距离，确定谁上、怎么上。

SBC｜尖刀-底盘耦合

先稳底盘（P25 过线）→ 再点尖刀（Top-2 覆盖）→ 才允许提速。

OPA｜组织相位对齐

用“角色边界剧本 / 接口剧本 / 决策 SLA”把“强什么的人”放到“需要什么的位置”。

RCP｜红线约束

底盘不过线（节奏/闭环/合规/稳态），暂停扩张。

R 指数（Resonance Index）

口径：R = A_pr^α × A_tb^β × Penalty^γ

  • A_pr：个人-角色适配（cos 相似 / 规范化相似）
  • A_tb：团队-项目吻合（0.6×平均对齐 + 0.4×Top-2 覆盖）
  • Penalty：红线惩罚（P25 未过线则指数衰减）
  • 默认权重：α=0.4、β=0.4、γ=0.2（阶段可调）

灯塔：R ≥ 0.75（绿：放量）｜0.60–0.75（黄：微调）｜<0.60（红：止血/换人/调项目）

示例数据与计算（看懂数字从哪里来）

样例 A（订阅 B2B）：

  • 本月新增付费 100，投放 ¥50,000 → CAC = 50,000 / 100 = ¥500
  • ARPU = ¥150/月，毛利率 75% → 月毛利 = 150 × 0.75 = ¥112.5
  • 回收期 = 500 / 112.5 = 4.44 个月（健康 ✅）
  • 订阅平均保持 18 月 → LTV = 150 × 0.75 × 18 = ¥2,025 → LTV/CAC = 2025/500 = 4.05（✅）
  • 期初存量收入 100k；扩张 +20k；缩减 -5k；流失 -10k → 期末 105k → NRR = 105/100 = 105%（需提升）
  • 应收账款 1,680,000；日均收入 30,000 → DSO = 1,680,000 / 30,000 = 56 天（可优化）

→ 结论：回收/LTV 健康；NRR 与 DSO 是下阶段重点；动作：升级演示模板化、年付折扣、催收 SOP。

6) PTJ 联动与三阶共振（含示例计算）

目标：预测“这支队在这个项目上能不能跑”。

R 指数示例（一步步算）

已知：

  • 个人-角色适配 A_pr = 0.82（个人雷达与该里程碑权重的余弦相似）
  • 团队-项目吻合：平均对齐 0.75；Top-2 覆盖 0.80 → A_tb = 0.6×0.75 + 0.4×0.80 = 0.77
  • 红线惩罚：P25(节奏)=58（阈 60），缺口 2 分 → 设 Penalty = exp(−λ×缺口)，取 λ=0.05 → Penalty = exp(−0.1)=0.905
  • 权重：α=0.4、β=0.4、γ=0.2

计算：R = 0.82^0.4 × 0.77^0.4 × 0.905^0.2 ≈ 0.77（绿区：可放量）

如果先把 P25(节奏) 拉到 61，Penalty≈exp(0)=1 → 新 R ≈ 0.85（更稳）。

8) 模型能干哪些“牛逼的事”（10 个扩展方向）

  1. 角色编排引擎：自动给里程碑配拍板/桥梁/稳盘与“角色边界剧本”。
  2. 任务编组·作战蜂群：3–5 人蜂群两周打点，复盘入“异常词典”。
  3. 客户派单与交付匹配：按客户画像派“桥梁+影响”组合并交付成功路线图。
  4. 招聘/面评配额：雷达凹槽→ JD 权重与面评 Rubric。
  5. 培训跃迁路径：弱边+相邻强边，4–6 周可验证跃迁。
  6. 激励与绩效分配：Top-2 覆盖杠杆→短期激励；底盘红线为前置。
  7. 并购整合雷达：两队叠加，30–60 天整合矩阵。
  8. 事故应急与韧性演习：on-call 地图+桌面演练+24h 无责复盘。
  9. 创新孵化 Pods：创新+推进双核，一周一原型，两周不通即“杀死/转向”。
  10. 远程健康护栏：静默带+接力棒+材料先行，减少夜会内耗。

9) 自证价值与 30–45 天拿单路径

唯一切入场景（建议）

在营加速器/孵化器团队（7–15 人）或 客户成功（CS）团队：决策快，对节奏/交付敏感，最容易两周见效。

三页补强（对投资与客户都有效）

  • ICP 与切入剧本：三问诊断（卡点/拍板/怕错）→ 承诺“三动作”→ 预约 D0/D1
  • 样板一页纸（模板）：现象 → 三动作 → 过程证据 → 体感变化
  • 报价与签约路径：首诊包（¥9,800/团队）→ SOP → 嵌入 → API；可附成功费

10) 小团队与家族私企微场景（文字即可落地）

  1. 亲戚当主管、团队不服：岗位边界上墙；验收按 DoD，不讲资历。
  2. 厂里小山头：接口剧本+接口台账，借人借料“走卡不走人情”。
  3. 财务在家里手里：用钱顺序表；先小单再放量。
  4. 老销售口头承诺：承诺白名单；先演最短路径，口头承诺归零入价目表。
  5. 生产小毛病多：三条红线+DoD 过表；高频问题→“异常词典”。
  6. 高学历经理与老班底冲突：经理做示范流程，老班底盯交付；规矩沉淀。
  7. 货品过杂仓库爆：冻结清单+快慢货墙；月度生杀会。
  8. 老板改主意频繁：一页北极星+不做清单；本周范围冻结。
  9. 二代接班：双闸门（质量/现金）+试验田+周汇报三件事。
  10. 采购回扣风险：三单对比+双人复核；月度公开抽检。

13) 附录：指标公式 · 示例数据表 · JSON 键名

关键公式（可复现）

CAC = 获客总花费 / 新增付费数
回收期（月） = CAC / (ARPU × 毛利率)
LTV = ARPU × 毛利率 × 订阅月数
LTV/CAC ≥ 3 → 健康
DSO(天) = 应收账款 / 日均收入
NRR = (期末存量收入 - 缩减 + 扩张) / 期初存量收入
R = A_pr^α × A_tb^β × Penalty^γ
Penalty = exp( -λ × max(0, 红线阈值 - P25缺口) )
A_tb = 0.6×cos(团队均值, 项目基线) + 0.4×coverage(Top-2, 项目杠杆)
